ARRAY METHODS FOR DUMMIES

Given an array:

[ 1 , 2 , 3 , 4 , 5]

When we pass pop method to the array:

const popMethod = (elements) => {

if (elements.length === 0) {

return undefined;

}

const lastElement = elements[elements.length - 1];

elements.length -= 1;

return lastElement;

};

Then it returns:

[5]
